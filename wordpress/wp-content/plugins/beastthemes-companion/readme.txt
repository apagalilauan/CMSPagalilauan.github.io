== Description ==

The beastthemes companion plugin adds extra functionality to the themes of beastthemes.

The beastthemes companion plugin features include:

- Beautiful ready-made homepage Sections
- Drag and drop page customization
- Predefined content sections
- Live content editing 
- And many other features
